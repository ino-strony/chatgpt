# Biblioteka

Samodzielna wtyczka WordPress rejestrująca typ wpisu **biblioteka** wraz z polami dodatkowymi oraz taksonomiami „kategorie” i „przedział wiekowy”. Zawiera gotowe szablony archiwum i pojedynczego wpisu, przygotowane do własnego ostylowania w motywie.

## Funkcje
- Własny typ wpisu **biblioteka** dostępny w panelu administracyjnym.
- Dodatkowe pola: imię i nazwisko autora, data wydania pierwszej edycji, data wydania bieżącej edycji, sugerowana cena brutto.
- Taksonomie: **kategorie** (hierarchiczna) oraz **przedział wiekowy** (tagi).
- Gotowe szablony `archive-biblioteka.php` i `single-biblioteka.php` używane automatycznie po aktywacji wtyczki.

## Instalacja i użycie
1. Skopiuj katalog `biblioteka` do katalogu `wp-content/plugins/` i aktywuj wtyczkę w kokpicie WordPressa.
2. W menu „Biblioteka” dodaj nowe pozycje, uzupełniając pola dodatkowe w metaboxie „Informacje o publikacji”.
3. Przypisz taksonomie „kategorie” i „przedział wiekowy”, aby filtrować zawartość na archiwum.
4. Odwiedź archiwum `/biblioteka/` lub pojedynczy wpis, aby skorzystać z dołączonych szablonów gotowych do ostylowania.
