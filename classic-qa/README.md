# Classic Q&A Accordion

Samodzielna wtyczka WordPress tworząca prosty moduł Q&A z klasycznym akordeonem na stronie.

## Funkcje
- Własny typ wpisu **Q&A** w panelu administracyjnym (tytuł = pytanie, edytor = odpowiedź).
- Pole "Kolejność" (menu order) dla łatwego sortowania.
- Shortcode `[classic_qa]` wyświetlający listę pytań z ikoną "+" i rozwijaną odpowiedzią.
- Lekka stylizacja i prosty skrypt jQuery (każde pytanie rozwijasz niezależnie).

## Instalacja i użycie
1. Skopiuj folder `classic-qa` do `wp-content/plugins/` i aktywuj wtyczkę.
2. W menu "Q&A" dodawaj nowe pytania i odpowiedzi w edytorze metaboxu.
3. W treści strony lub wpisu wstaw shortcode `[classic_qa]` (opcjonalnie `order="DESC"` dla odwróconej kolejności).
