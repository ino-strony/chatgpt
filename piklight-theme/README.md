# Pik-Light Custom Theme

Motyw WordPress przygotowany na podstawie załączonego projektu graficznego. Zawiera edytowalny nagłówek i stopkę w Customizerze oraz dynamiczne bloki Gutenberg dla głównych sekcji strony.

## Bloki Gutenberg

Po aktywacji motywu w edytorze blokowym dostępne są bloki:

- Hero Pik-Light
- Atuty Pik-Light
- Oferta Pik-Light
- O firmie Pik-Light
- Polecane produkty Pik-Light
- Współpraca hurtowa Pik-Light

Każdy blok zapisuje dane jako atrybuty Gutenberga i renderuje sekcję dynamicznie po stronie PHP.

## Edycja headera i footera

Dane nagłówka oraz stopki można zmieniać w panelu WordPress: **Wygląd → Personalizacja**.

Dostępne ustawienia obejmują logo, telefon, CTA, opis stopki, adres, e-mail, telefon, katalog PDF i copyright.
